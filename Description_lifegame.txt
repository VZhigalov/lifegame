Запуск:
Нажмите по иконке "graph.py", либо введите в командной строке "python graph.py"

Использование:
После запуска программа попросит ввести размер поля для игры, то есть число, представляющее ширину и высоту.
Затем появится окно с квадратным полем для игры введенных размеров и кнопкой "Start". "Живые" клетки черного цвета, "мертвые" - белого. 
Обозначьте стартовые позиции "живых" клеток нажатием на них мышкой и нажмите кнопку "Start". Выполнится первая итерация игры.
После каждого нажатия на "Start" выполняется очередная итерация.
